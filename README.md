## Project Overview
VeriAfrica is a mobile-first app designed to provide African users with verified news and insights. Features include:
- Africa Now: verified local news
- Global Pulse: curated global news with African context
- Viral Claims: trending X/Twitter claims with AI verification
- Fact Check: AI-powered claim analysis
- Opportunities: scholarships, programs, fellowships

## Tech Stack
- Frontend: Windsurf (low-code)
- Backend: Serverless APIs / AI scripts
- AI/NLP: OpenAI API for summarization and verification
- Database: Windsurf collections or serverless DB
